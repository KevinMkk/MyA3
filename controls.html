Sky Kingdom VR

🧭 Movement Controls

  • W / A / S / D → Move forward / left / backward / right
  • Arrow Keys (↑ ↓ ← →) → Alternative movement control mode
  • Mouse → Look around (camera rotation)
  • Shift → Sprint (hold for faster movement)
  • Right Shift → Toggle control mode:
    • Arrow Keys = Turn + Move Forward
    • OR Arrow Keys = Strafe Left/Right
  • Space → Jump / rise into air
  • C → Descend / fly down

☁ Cloud Navigation System

The world is structured as floating cloud regions. Each major destination exists in a directional cloud zone:

  • Bokong Nature Reserve → WEST clouds
  • Ts'ehlanyane National Park → SOUTH clouds
  • Afriski → EAST clouds

You must fly or walk into the correct cloud region to reach teleport zones.

🗺 Map System

  • M → Open / Close Map
  • Map shows real-world locations in Lesotho
  • You can either:
    • Instant teleport (if enabled)
    • Receive directional guidance
  • Map pauses the game and frees the mouse

🎮 Gameplay Systems

  • Teleport Zones → Floating cloud portals in world
  • T → Return to start location
  • F → Interact with doors/objects
  • Game can switch between:
    • Instant teleport mode
    • Navigation guidance mode

⌨️ ESC & Menu System

  • ESC → Opens and closes the in-game menu
  • ☰ Menu Button → Same function as ESC
  • When the menu is opened:
    • Mouse cursor is unlocked for UI interaction
    • The player can interact with menu options
    • Camera control is temporarily released from mouse input
  • When the menu is closed:
    • Mouse cursor is locked back to the game
    • Camera control resumes normally

📋 In-Game Menu Options

  • Controls → Opens detailed movement and gameplay instructions
  • Map → Opens world map showing locations in Lesotho
  • Teleport Mode → Switches between:
    • Instant Teleport → Select location to travel immediately
    • Direction Mode → Shows cloud navigation instructions instead of teleporting
  • Restart → Reloads the current scene from the beginning
  • Exit → Returns to the main menu screen
