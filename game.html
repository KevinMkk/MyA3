Menu

Select a location marker.

Controls

Movement:
W / Q / S / E or Arrow Keys → Move
A / D or ← / → Arrow Keys → Rotate / Look
Mouse → Rotate / Free Look

Space → Jump / Fly Up
C → Fly Down

Modes:
Hold Shift → Sprint
Right Shift → Toggle Arrow Mode (Rotate vs Strafe)
Double Space → Fly

V → Camera Switch

M → Open Map
F → Use Doors
T → Return To Start

ESC → Open / Close Menu

Map:
Click Markers → Teleport To Location
Toggle Teleport Mode → Switch Between Instant Teleport and Guide Mode (Shows Path To Location)
